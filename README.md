# 漢字手書きテストシステム

マウスや指で漢字を書いて学習できるブラウザアプリです。  
出題された漢字と、自分が書いた漢字の形を比較して採点します。

## デモ
👉 [GitHub Pagesで開く](https://ユーザー名.github.io/kanji-write-quiz/)

## 特徴
- ブラウザだけで動作（インストール不要）
- マウス・タッチに対応
- 模範文字との形の一致度(IoU)で採点
- 出題リストをJSONで管理（自由に編集可能）
